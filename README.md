### Dependencies
The models has been tested in Google Colab which has Python Version of 3.10.12, with following required packages: <br>
1. torch==2.0.1
2. transformer==4.30.2
3. torchtext==0.6
4. numpy==1.22.4 
5. pandas==1.5.3
